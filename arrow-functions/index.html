Arrow Functions

New with JavaScript since ES6 / ES2015, arrow functions, also known as fat arrow functions, are a concise way to write function expressions.

An arrow function expression is a syntactically compact alternative to a regular function expression, although without its own bindings to the this, arguments, super, or new.target keywords. Arrow function expressions are ill suited as methods, and they cannot be used as constructors.

Arrow Functions: 

//with parameters
let myFunc = (a, b) => {
    return a * b;
};

//with 1 param
let myFunc = a => a * a;

//object literal
let myFunc = (first, last) =>
({ firstName: first, lastName: last });

Arrow functions can be used with no params,1 or more params, object literal, iterate array using arrow functions etc. Please check JavaScript file for more details
