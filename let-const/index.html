Let & Const

The let statement declares a block scope local variable, optionally initializing it to a value.
Constants are block-scoped, much like variables defined using the let statement. The value of a constant can't be changed through reassignment, and it can't be redeclared.

Let & Const: let allows you to declare variables that are limited to a scope of a block statement, or expression on which it is used, unlike the var keyword.With const you can define immutable variables (constants). Trying to re-assign a constant will raise an error.

ES5 Variable(var) 

var dog = 'Ralf';

if (true) {
    var dog = 'Skip';
}

console.log(dog); // Skip

ES6 let keyword 

let dog = 'Ralf';

if (true) {
    let dog = 'Skip';
}

console.log(dog); // Ralf

ES6 const keyword 

const someArr = [3, 4, 5];

someArr.push(6);// [3,4,5,6]

someArr = [10]; //Error: "someArr" is read-only

const deals differently with arrays and objects. You can insert new values to arrays/ojects but cannot re-assign. Please check JS file.
