Default Params

Simple and intuitive default values for function parameters

You can now define default values for your function parameters in JavaScript. The default value will be used when an argument is missing or it evaluates to undefined.

Default Params: 

function add(x, y = 3) {
    console.log(x + y);
}
    
add(3, 9); // 12
add(3) // 6
add(12, undefined) // 15
add(undefined, 8); // NaN, x doesn't have a default value

Default parameters can be really useful to ensure you have at least an empty array or object literal available when performing operations. Please check JavaScript file for an example with/without default parameter
