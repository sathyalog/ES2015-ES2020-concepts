For in Loop

Eventhough For-in-loop is not introduced in ES6, it could be better to understand this iterator.
Use for…in to iterate over the properties of an object (the object keys)

Using for-in-loop: 

let oldCar = {
    make: 'Toyota',
    model: 'Tercel',
    year: '1996'
};

for (let key in oldCar) {
    console.log(`${key} --> ${oldCar[key]}`);
}

For-in-loop iterate over the properties of an object (the object keys).You can also use for…in to iterate over the index values of an iterable like an array or a string.
For..of Vs For..in
Use for…of to iterate over the values in an iterable, like an array
Use for…in to iterate over the properties of an object(Object keys)
