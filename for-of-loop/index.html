For of Loop

Before ES5, developers iterate using
for(i='0';i<'2';i++)
With ES5, forEach introduced and in ES6 for-of introduced.
The for...of statement creates a loop iterating over iterable objects, including: built-in String, Array, array-like objects (e.g., arguments or NodeList), TypedArray, Map, Set, and user-defined iterables.

Using ES5 forEach 
const incomes = [65000, 85000, 120000];
incomes.forEach(function (value) {
    console.log(value);
});

Using for-of-loop(ES6): 

for(income of incomes) {
    console.log(income);
}

For-of-loop can be used to iterate arrays, strings, maps, sets. Please check JavaScript file for more details
