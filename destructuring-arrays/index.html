Destructuring Arrays

Destructuring simply implies breaking down a complex structure into simpler parts.

In JavaScript, this complex structure is usually an object or an array. With the destructuring syntax, you can extract smaller fragments from arrays and objects. Destructuring syntax can be used for variable declaration or variable assignment.

Basic Destructuring, Declaring Variables before Assingment, Skipping Items in an Array, Assigning the rest of an array and Destructuring Assignment with Functions are few concepts in array destructuring.
