Classes

Class: It is the before the plan of creating any objects which are known as blueprint of any objects which you want to create.

ES5 Constructor function: 

// ES5 Constructor Function
function Person(name) {
    this.name = name;
}

// ES5 adding a method to the Person prototype
Person.prototype.walk = function() {
  console.log(this.name + ' is walking.');
}
            
var employee = new Person('Sathya');
console.log(employee.name);  // Outputs 'Sathya'
employee.walk(); // Outputs 'Sathya is walking.'

ES2015/ES6 Classes: 

class Person {
    constructor(name) {
        this.name = name;
    }
    get fname() {
        return this.name.toUpperCase();
    }
    walk() {
        console.log(this.name + ' is walking.');
    }
    static test() {
        console.log('static keyword helps to call a method from class without even instantiaion')
    }
}

Person.test(); //static keyword helps to call a method from class without even instantiaion             
let employee = new Person('Sathya');
employee.walk(); Outputs 'Sathya is walking.'
console.log(employee.fname); Outputs SATHYA

ES6 classes brings a new syntax for getters and setters on object properties. Get and set allows us to run code on the reading or writing of a property.
Using static can allow to access the method without instantiation. For more verify JS file.
To access or modify a property we would use a getter (access) or a setter (modify), which are specific methods we define in our class.
