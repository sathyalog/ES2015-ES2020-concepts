Sets

Sets are a new object type with ES6 (ES2015) that allow to create collections of unique values. The values in a set can be either simple primitives like strings or integers, but more complex object types like object literals or arrays can also be part of a set.

Hereâ€™s a simple example showing off a basic set and a few of the available methods on it like add, size, has, forEach, delete and clear

Sets: 

const exampleSet = new Set([1,1,1,1,2,2,2,2]);
console.log(exampleSet.size);//2
console.log(exampleSet.add(5).add(17)); //Set(4) [ 1, 2, 5, 17 ]
console.log(exampleSet.size); //4
console.log(exampleSet.has(5)); //true
console.log(exampleSet.delete(5));
console.log(exampleSet.size); //3
exampleSet.clear(); 
console.log(exampleSet.size); //0
