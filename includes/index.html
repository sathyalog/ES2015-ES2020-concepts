Array Includes

It determines whether an array contains a given element and returns either true or false.

ES6 (ECMAScript 6) introduced Array Includes as a replacement for indexOf method in JavaScript.

Array Includes: 

let numbers = [1,2,3,4,5,6];

numbers.includes(0) // false
numbers.includes(4) // true

ES5 indexOf method either returns -1 or its index position. This ES6 includes will return either true or false.
